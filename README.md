# 📋 To-Do List (Python CLI)

## 📌 Overview
This is a simple command-line To-Do List application built with Python.  
It allows users to manage their daily tasks by adding, viewing, and removing items from a list.

---

## ✨ Features
- ✅ Add new tasks
- ✅ View all tasks
- ✅ Remove existing tasks
- ✅ Continuous loop for repeated use
- ✅ Basic error handling (task not found)

---

## 🛠️ Tech Stack
- Python (no external libraries required)

---

## ⚙️ How It Works
The program runs in a loop and gives the user 3 options:
1. Add a task
2. View tasks
3. Remove a task  

After each action, the user can choose to continue or exit.
